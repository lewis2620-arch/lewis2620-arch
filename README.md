## Governance Frameworks

Two original governance standards I've built and use in every agentic system:

**[AQGP — Agentic Quality & Governance Protocol](./skills/original-agentic/E-governance/53-AQGP-agentic-quality-governance-protocol.md)**
Universal governance floor for all multi-agent systems. Agent accountability, proof-before-status, inter-agent trust rules, escalation triggers, graceful degradation. Every fleet I build runs this.

**[CVAP — Clinical Verified Action Protocol](./skills/original-agentic/E-governance/52-CVAP-clinical-verified-action-protocol.md)**
Clinical extension of AQGP for regulated environments. Built on a core principle: a false completion in a clinical AI system is a patient safety event, not a UX failure.

```
AQGP (Universal floor — all multi-agent systems)
  └── CVAP (Clinical/regulated extension)
```
